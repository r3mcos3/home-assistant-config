## Hardware

I'm running Home Assistant from my old desktop pc, It's a
Intel Core I5 2500K with 8GB ram and a 500GB ssd.
Its running in a vm on Proxmox.

Other hardware.

* [Conbee II](https://www.phoscon.de/en/conbee2) Stick
* [Ikea Tradfri](https://www.ikea.com/) Lights
* [Ikea Tradfri](https://www.ikea.com/) Fytur roller blind
* 2x [Google Nest Mini](https://store.google.com/nl/product/google_nest_mini)
* [Google Nest Hub](https://store.google.com/nl/product/google_nest_hub)
* [Sonoff](https://www.itead.cc/) Devices (Pow R2, Basic, Tx)
* Various [Xiaomi](https://www.xiaomiproducts.nl/smart-home/) Sensors (door/window, motion, temp/hum/press)
* Various other tuya based lights flashed with [Tuya Convert](https://github.com/ct-Open-Source/tuya-convert)
